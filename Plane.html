Plane Model

2/21/17

Assignment

For this project, create a model of an airplane using polygonal modeling. Try to take advantage of the extrusion, bevel, poke, taper, etc., features in Maya. The textbook contains a tutorial on how to construct such a model in Chapter 6, which you may use for your project.

Description

The plane modeled below is not based on an real world design. I doodled it myself. My apologies if it does not make any sense from an aeronautic engineer's perspective. I would be surprised if anyone decided to make a life sized replica, so I think we will be alright. My initial sketch is given below as well as the model from multiple different perspectives and a video giving a 360 degree display.
